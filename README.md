## 一、世界观

故事发生在一个**由古代人(旧人类)用顶级科技亲手创造**的世界。当末日不可逆地临近，他们将一种本不属于此处的能量 —— **能量 X** —— 引入这片大陆，希望让后来者用魔法对抗 "世界的终焉"。

但他们的设想并不完美。

### 1. 两种魔法

世界上同时存在两种本质不同的魔法体系。它们对**能量 X** 的处理方式相反，也成为整个文明分裂的根源。

#### 魔法 A — "现代正统"

> 吸引环境中的能量 X → 聚集为魔力因子 A → 融合 → 储存 → 编排（体内 / 体外）→ 释放

- 现代社会主流，被认为是**安全、可控、可发展**的魔法
- 强者往往在 "融合 / 储存 / 编排 / 释放" 某一环节天赋异常
- 体内编排决定**属性**（元素 / 空间 / 诅咒），体外编排决定**输出方式**（直击 / 阵术 / 依附）
- 释放后，魔力因子最终会回归能量 X 形态，**对环境无永久副作用**

#### 魔法 B — "古代正统 / 现代异端"

> 捕捉、控制环境中的魔力因子 → 排列 → 失控分解 → 爆发

- 古代人留给本世界的"原始遗产"，本应是**对抗终焉的真正手段**
- 由强烈情感与巨大能量驱动，威力远超魔法 A，持续时间更长
- 但失控分解会产生**永远不会回归能量 X** 的魔力因子 B，扭曲所在环境的本质
- 当代被定性为**邪教 / 异端魔法**，记载它的古代史料被视为禁书
- 真相只在少数派别手中

#### A 与 B 的核心矛盾

| | 魔法 A | 魔法 B |
|---|---|---|
| 副作用 | 主要是攻击波及 | **永久扭曲环境本质** |
| 能量消耗 | 低 | 极高 |
| 持续时间 | 短（除非特殊编排） | 通常很长 |
| 控制难度 | 高 | 较低，但极易失控 |
| 大众认知 | 正统 | 异端 |
| 真实性质 | 后来者的发明 | 古代人留下的真正方案 |

### 2. 三种科技

- **科技 A** —— 不依赖任何魔法的纯科技，已发展到可与小型魔法抗衡，但远不及古代人巅峰
- **科技 B** —— 以魔法 A 为底层的混合科技，本世代发展前景广阔，但被部分高层抵制
- **科技 C** —— 古代人遗留的科技顶点，远超现代任何技术；大部分已失传，被现代视为不存在的传说

### 3. 终焉与重启

世界本身存在一个**不可避免的终焉**。古代人为延续文明，建立了"重启"机制（细节由教派 B 各派别理解不同）。围绕"是否对抗终焉 / 是否承认重启 / 魔法 B 是否该被复兴"，整个大陆被切成数个互相博弈的阵营。

## 二、世界格局

大陆按"科技 A : 科技 B : 魔法 A"的占比可粗略分为四类：

- **均衡型（1 类）**：发展全面，例如 1 号 A 国（科技强国 + 一定魔法基础）
- **科技独尊型（2 类）**：拒绝魔法，例如 2 号 A 国 / 2 号 B 国
- **魔法 / 自然崇拜型（3 类）**：科技 B 主导，几乎不发展纯科技，例如 3 号 A 国
- **魔法集落（4 类）**：由具备完整魔法 A 能力的人组成的城邦，与教派 B 关系密切

### 主要势力速览

| 势力 | 立场 |
|---|---|
| **1 号 A 国** | "异常消除组织" 总部所在地，对外统一、内斗严重；信仰风与雨 |
| **2 号 A 国** | 古代 AI 大量隐藏其中，借邪教与魔法集落各派内斗谋势，**不关心终焉** |
| **3 号 A 国** | 自然 + 科技 B 双重崇拜，将 1 号 A 国视为最大政治对手 |
| **2 号 B 国** | 大陆最北，亚人科技强国，秘密研究"科技+魔法 B"的超大型战争兵器 |
| **4 号 B 国** | 隐秘的亚人魔法国，掌握部分被脱出的古代信息 |
| **魔法集落** | 高魔素纯魔法 A 城邦，是众多关键派别的根据地 |
| **教派 A** | 利用魔法 B 的邪教，掌握片面古代信息 |
| **教派 B** | 真正的古代史料传承者，内部分裂为复兴派 / 重启崇拜派 / 弃绝派三大对立派别 |
| **魔法协会** | 现代成立，主导魔法 A 研究但对真正本源所知有限 |
| **异常消除组织** | 跨国应对失控魔法 B 现象的组织，各国动机各异 |

### 中立地带

世界上散落着多块**不属于任何国家的中立地带**：山脉、荒漠、森林、人鱼村、深渊高塔、圣湖、断崖瀑布、冰原 …… 这些地方往往藏着关于古代人、关于"柱"、关于真相的碎片。

## 三、种族

### 现代人种

- **XX 人 / zz 人 / cc 人 / tt 人 / ss 人 / Yy 人** ：分别擅长诅咒、咒术、元素、科技、聚集科技、空间魔法等不同方向
- 即使无法使用魔法，现代人种的身体素质也普遍强于古代人

### 亚人种

龙种、曜人种（白昼独行 / 母系 / 与古代禁忌相关）、雪兽种、蛙种、兔种、狐种、狼种、羊种、天梭种、鸟种、蛛种、式终种、人鱼种 ……

每个亚人种有独特的体貌、生态与魔法适性。

### 其他生命

- **精灵** ：X 世纪后由魔力因子奇异组合诞生的生命体，智力低但力量强
- **魔物** ：被能量 X "唤醒"获得意识的动物或植物
- **古代人 / 古代 AI** ：少数仍以网络意识、机械之躯存活于现代，平静地等待终焉

## 四、表达方向

游戏想做的事，按优先级：

1. **2.5D 动作手感** —— 横版战斗为主，关键场景切换至俯视；战斗追求帧级反馈、连段、闪避
2. **第四面墙叙事** —— 不止于角色之间的对话，会以系统层面的方式与玩家互动
3. **多结局 / 循环** —— 每一次重玩都不只是 New Game+
4. **隐藏真相** —— 大量信息散落在中立地带、教派文献、亚人种生态里，不强加，让愿意深挖的玩家发现
